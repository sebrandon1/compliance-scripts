## Overview

Additional SSHD hardening beyond M1 (E8). Sets idle session timeout and keepalive settings to terminate inactive SSH connections, reducing the window for session hijacking.

**Profile**: NIST 800-53 Moderate (`rhcos4-moderate`)

## Compliance Checks

| Check | Description |
|-------|-------------|
| `ClientAliveInterval` | Set SSH idle timeout interval |
| `ClientAliveCountMax` | Set SSH keepalive count before disconnect |

## Verification

```bash
oc debug node/<node> -- chroot /host sshd -T | grep -E 'clientaliveinterval|clientalivecountmax'
```
